# Seja bem-vindo ao meu Portfólio! 😉

Este projeto foi desenvolvido utilizando as seguintes linguagens:
- HTML e CSS
- JavaScript

Nele constam, minhas skills, detalhes pessoais, tipos de serviços, minha linha do tempo, projetos destaque e avaliações de meus clientes.
É possível também encontrar links de acesso ao meu LinkedIn, Twitter e GitHub. Além da presença de um formulário de contato. Desenvolvi este Portfólio com o objetivo de agregar mais em meu conhecimento de programação, e poder demonstrar minhas habilidades atuais e que sempre farei de tudo para <b> ir além! </b>

Esta solução também é responsiva, sendo possível abri-lá em celulares, computadores ou notebooks. Todos os arquivos utilizados estão disponíveis neste repositório. <br>
